# Architecture

Plataforma Django con API REST (DRF) + Frontend HTML.

## Módulos
- **authentication**: Login + JWT + roles/permisos.
- **etl**: Engine ETL, carga a BD y historial ETL.
- **analytics**: Estadística descriptiva, KPIs y segmentaciones.
- **ml**: Entrenamiento y predicción (Random Forest) + métricas.
- **reports**: Reportes y exportaciones (CSV/Excel/PDF).
- **dashboard**: Página SPA/HTML para consumir APIs.
