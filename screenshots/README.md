Para reemplazarlas por capturas reales, ejecute el servidor local, ingrese al dashboard y use la herramienta de captura del sistema operativo o del navegador.

Se recomienda capturar al menos:

1. Dashboard principal con KPIs.
2. Carga manual de CSV/Excel.
3. Predicción ML individual.
4. Exportaciones desde Dashboard o Analítica.
5. PDF generado desde Reportes.
6. Documentación API en `/api/docs/`.
7. Métricas ML en Django Admin.
